## Обзор

### Задача

Создать приложение для отслеживания IP-адресов, максимально приближенное к дизайну, находящемуся в папке `/design`. Дизайн представлен в статическом формате JPG. Использование JPG-файлов потребует самостоятельного выбора стилей, таких как `font-size`, `padding` и `margin`.

Все необходимые ресурсы находятся в папке `/images`. Ресурсы уже оптимизированы. Имеется также файл `style-guide.md`, содержащий необходимую информацию, такую как цветовая палитра и шрифты.

Для получения местоположения IP-адресов необходимо использовать [API геолокации IP от IPify](https://geo.ipify.org/). В остальном можно использовать любые инструменты, которые нравятся, чтобы выполнить задачу.

Пользователь должен иметь возможность:

- просматривать оптимальную компоновку каждой страницы в зависимости от размера экрана их устройства;
- видеть состояния при наведении курсора на все интерактивные элементы на странице;
- искать любые IP-адреса или домены и видеть ключевую информацию и местоположение.

## Быстрый старт

### Предварительные требования

- **Node.js** ≥ 18
- npm, yarn или pnpm

### Установка и запуск

1. Клонируйте репозиторий

```bash
# 1. Клонируйте репозиторий
git clone https://github.com/zendrolya/ip-address-tracker.git
cd ip-address-tracker
```

2. Установите зависимости

```bash
npm install
```

3. Создайте файл .env и добавьте API-ключ IPify

```bash
# отредактируйте .env, вставив свой ключ:
VITE_GEO_IPIFY_API_KEY=ваш_ключ
```

4. Запустите dev-сервер

```bash
npm run dev
```

### Сборка для продакшна

```bash
npm run build
```

Результат будет в папке `dist/`.

### Деплой на GitHub Pages

```bash
npm run deploy
```

Скрипт собирает проект и публикует содержимое `dist/` в ветку `gh-pages`.

## Как работает приложение

1. **При загрузке** страницы инициализируется карта MapLibre с видом на Москву (по умолчанию).
2. **Пользователь вводит** IP-адрес в форму поиска.
3. **Валидация**: `validate-ip.js` проверяет адрес регулярным выражением для IPv4.
4. **API-запрос**: при корректном IP выполняется запрос к `geo.ipify.org/api/v2/country,city` для получения данных о локации.
5. **Отображение результатов**: IP, регион, город, почтовый индекс, таймзона и ISP выводятся на страницу.
6. **Карта обновляется**: функция `flyTo()` плавно перемещает камеру к найденным координатам, маркер устанавливается на место.

```
.header

.header__btn
.info__element
.info__element-title
.info__element-text
.footer__attribution
.footer__link
```

### Валидация IP

Функция `validateIp()` проверяет введённое значение на соответствие формату IPv4. При некорректном вводе пользователь получает уведомление через `alert()`.
